# Darts-game

Prompt: Darts Puzzle Journey App (Candy Crush Style)

Build a mobile-first casual game app called “Checkout Quest” — a darts-based puzzle progression game inspired by Candy Crush level structure.

Core Concept

The game is a level-based journey where players complete increasingly difficult darts “checkout” challenges. Each level is a self-contained puzzle with specific rules and constraints. Players progress through a map of levels, unlocking new stages after completing each challenge.

Gameplay Loop
Each level presents a checkout target (e.g. 66, 80, 123, 170, etc.)
The player has a limited number of darts per level (e.g. 2, 3, or 6 darts)
The goal is to finish exactly at zero
If the player busts or fails to finish within dart limit → level failed
If successful → level completed and next level unlocked
Level Types (examples)

Include variety in level design:

Standard Checkout
Example: “Checkout 66 in 3 darts”
Limited Dart Challenge
Example: “Checkout 80 in 2 darts only”
Precision Constraint Levels
Example: “Checkout 50 using ONLY doubles”
Example: “Finish 100 using at least 1 treble”
Route Puzzle Levels
Example: “Checkout 123 with last dart being a double”
Hard Mode Milestones
Example: “Checkout 170 in 3 darts (pro level)”
Progression System
Map-style level progression (Candy Crush style)
Levels grouped into “boards” or “chapters”
Increasing difficulty over time
Stars or ratings system:
1 star = complete
2–3 stars = efficiency (few darts used / perfect finish)
Scoring System
Bonus points for:
finishing in fewer darts
hitting doubles/trebles when required
“checkout accuracy” (clean finish vs messy finish)
Optional leaderboard support later
UX / UI Style
Bright, playful, arcade-style visuals (Candy Crush inspired)
Animated darts hitting a dartboard
Satisfying feedback for doubles/trebles/checkouts
Simple tap-to-aim or drag-to-aim mechanic (can be simplified for MVP)
Key Screens
Home screen (Start / Continue journey)
Level map (progression path)
Level gameplay screen (dartboard + remaining darts + target score)
Results screen (stars, retry, next level)
Optional: Practice mode (free play checkout trainer)
Extra Features (nice to have)
Daily challenges (random checkout puzzles)
“Boss levels” every 10–20 levels
Unlockable dart skins
Hints (suggest best finish route)
Important Design Goal

Make it feel like a puzzle game first, darts simulator second — quick levels, addictive progression, and satisfying problem-solving.
